# 🎨 Sketchbook Web Clone

**Fully functional, sleek & mobile-first digital sketching web application**

## 📖 Description
A beautiful, responsive recreation of the award-winning Sketchbook app. Draw, sketch, paint with natural brushes, infinite canvas, layers, color picker, undo/redo, eraser, smudge, guides and more — everything works instantly in your browser using HTML5 Canvas. Built exactly as requested with pure HTML5, CSS3, JavaScript, jQuery & Bootstrap 5. No folders, no external links.

## ✨ Features
- 🖌️ Professional brush engine (Brush, Eraser, Smudge, Marker, Airbrush)
- 📱 100% responsive & touch-friendly (desktop, tablet, mobile phones & all devices)
- 🧱 Infinite canvas with real-time drawing
- 🎨 Live color picker + quick swatches
- 📏 Adjustable brush size with live preview
- 🔄 Undo / Redo stack (30 steps)
- 🧩 Fake Layers panel with visibility & opacity
- 📐 Toggleable guides & rulers
- 🌟 Elegant dark UI with signature green accents
- 💾 Export-ready canvas (PNG ready in real use)
- 🚀 Zero external dependencies beyond CDNs

## 🛠️ Technologies
- **HTML5** – Canvas + semantic markup
- **CSS3** – Modern gradients & smooth animations
- **JavaScript** – Full drawing engine & touch support
- **jQuery** – Easy UI interactions
- **Bootstrap 5** – Responsive layout & components

## 🚀 How to Run
1. Download the four files (`index.html`, `style.css`, `script.js`, `README.md`)
2. Place them together in one folder (no sub-folders)
3. Open `index.html` in any modern browser
4. Start drawing immediately with mouse or finger!
